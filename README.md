# BizDrive Manager

> 차량 운행 관리 시스템 - Premium Dark Edition

## ✨ 주요 기능

### 📊 사용자 기능
- **대시보드**: 차량별 통계 및 예약 현황
- **차량운행기록**: 상세 운행 내역 관리 (📸 사진 촬영 지원)
- **차량예약**: 캘린더 기반 예약 시스템
- **주유 및 톨비기록**: 비용 추적
- **차량주차위치**: 실시간 주차 위치 확인
- **정비기록**: 차량 정비 이력 관리
- **자주가는곳**: 목적지 및 주차장 즐겨찾기
- **사용설명서**: 아코디언 뷰
- **피드백**: 개선 제안 제출

### 🔧 관리자 기능
- **차량관리**: 차량 등록/수정/삭제
- **사용자관리**: 권한 및 계정 관리 (초대 기능)
- **회사관리**: 기업 정보 설정

## 🎨 디자인
